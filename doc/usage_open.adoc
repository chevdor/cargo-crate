crate-open 0.0.1

Opens the crate in a browser

USAGE:
    crate open [OPTIONS] <CRATE_NAME>

ARGS:
    <CRATE_NAME>    The name of the crate to open in your browser

OPTIONS:
    -h, --help       Print help information
    -j, --json       Output as json
    -V, --version    Print version information
